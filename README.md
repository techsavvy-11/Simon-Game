# Simon-Game

Classic Simon memory game built with vanilla HTML, CSS, and jQuery. Test your memory by repeating increasingly complex sequences of colored buttons and sounds.

  4 colored buttons (Red, Blue, Green, Yellow) with unique sounds

  Progressive difficulty - sequences get longer each round

  Visual button press animations and audio feedback

  Keyboard start (any key press begins first sequence)

  Game over detection with restart functionality

  Responsive design with CSS hover/game-over effects

How to Play

  Press any key to start

  Watch/listen to the sequence

  Click the colored buttons in the exact order shown/heard

  Repeat with longer sequences until you make a mistake

  Press any key to restart after game over
